Skilled Sapiens LMS - Hostinger Deployment Package
Release: 20260718-certificate-direct-download

What this package contains
- public_html/: complete latest production frontend build for Hostinger.
- public_html/.htaccess: SPA routing support for direct route refreshes.
- public_html/assets/: hashed JavaScript and CSS bundles generated by Vite.
- public_html/favicon.ico and related icon files.

Latest focused change included
- Student certificate download now downloads the certificate PDF directly from the portal CTA.
- The student should not be redirected to the long Supabase signed URL page.
- The admin certificate issuance flow and Supabase Edge Function behavior are unchanged.

Deployment steps
1. Open this package ZIP.
2. Upload the contents inside public_html/ to the Hostinger public_html directory.
3. Replace existing frontend files when prompted.
4. Do not upload this guide file into public_html unless you specifically want it visible on hosting.

Verification after upload
1. Open the LMS login page.
2. Log in as a student with an issued certificate.
3. Go to Certificates.
4. Click the certificate download CTA.
5. Confirm the PDF downloads directly and the student remains on the LMS page.

Build verification before packaging
- npm run build: passed.

Supabase deployment
- Not required for this package.
